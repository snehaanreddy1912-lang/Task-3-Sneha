# Project 3 - SQL Data Analysis

## Objective

The objective of this project is to use SQL queries to extract meaningful insights from a business dataset. The project focuses on filtering, sorting, grouping, and aggregating data to support data-driven decision-making.

## Tools Used

* MySQL 8.0
* SQL
* GitHub

## Dataset Information

The dataset contains 1200 order records with information about:

* Order Details
* Products
* Customers
* Payment Methods
* Referral Sources
* Revenue

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* COUNT()
* SUM()
* AVG()

## Queries Performed

### Query 1: View All Records

Used SELECT to retrieve all records from the Orders table.

### Query 2: Filter High Value Orders

Used WHERE to display orders with TotalPrice greater than 1000.

### Query 3: Sort Orders by Total Price

Used ORDER BY to sort orders from highest to lowest value.

### Query 4: Count Orders by Product

Used GROUP BY and COUNT() to determine product-wise order volume.

### Query 5: Calculate Revenue by Product

Used SUM() to calculate total revenue generated by each product.

### Query 6: Calculate Average Order Value by Product

Used AVG() to calculate average order value for each product.

### Query 7: Analyze Payment Methods

Used GROUP BY and COUNT() to analyze customer payment preferences.

### Query 8: Analyze Referral Sources

Used GROUP BY and COUNT() to identify the most effective referral channels.
